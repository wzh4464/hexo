# 帮助对冲

## 策略

1. 当前汇率0.93, 每万元, 交9300RMB
2. 5日后, 若人民币上涨 比如到0.90, 我们退给她300RMB
3. 若人民币下跌, 则不需补差价。

## 结果

|                 | 上涨                  | 下跌                  |
| --------------- | --------------------- | --------------------- |
| 无策略，8.1结算 | 收9000rmb, 付10000hkd | 收9600rmb, 付10000hkd |
| 有策略          | 收9000rmb, 付10000hkd | 收9300rmb, 付10000hkd |
